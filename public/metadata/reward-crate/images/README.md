# NFT Images

This folder contains the images for the reward crate NFTs.

## File Structure:
- `0.png` - Standard tier image
- `1.png` - Ardent Fan tier image  
- `2.png` - Super Stellar tier image

## Image Requirements:
- Format: PNG
- Recommended size: 512x512 pixels or higher
- Square aspect ratio preferred
- High quality for marketplace display

## Usage:
These images are referenced in the corresponding JSON metadata files and will be displayed on NFT marketplaces and in wallets.

**Note:** Make sure to update the image URLs in the JSON metadata files to point to your actual domain/IPFS when deploying.
